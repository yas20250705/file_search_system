検索オプションの使い方
AND検索（デフォルト）: スペースで区切ると、全ての語を含むファイルを検索します。
例: python tutorial → pythonとtutorialの両方を含む
OR検索: OR または | で、いずれかの語を含むファイルを検索します。
例: python OR tutorial または python | tutorial
除外ワード: - プレフィックスで、特定の語を含まない結果を除外します。
例: python -tutorial → pythonを含むがtutorialを含まない
フレーズ検索: "..." で囲むと、語の順序を保持して検索します。
例: "machine learning"
厳密フレーズ検索: ""..."" で囲むと、完全一致で検索します。
例: ""exact phrase""
